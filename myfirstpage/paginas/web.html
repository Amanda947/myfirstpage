Matéria Programação Básica para Web
Carga horária pesencial 60
Ementa
  1. Construção de páginas Web por meio de Web Standards (W3C);
  2. Linguagem de marcação: HyperText Markup Language (HTML);
  3. Construção de estilos para páginas Web com Cascade Style Sheet (CSS);
  4. Introdução de comportamentos em páginas Web por meio de programação Javascript
Competências
  1. Conhecer fundamentos, tecnologias e ferramentas para desenvolver páginas Web;
  2. Desenvolver páginas Web de acordo com os padrões Web do W3C
Habilidades
  1. Utilização de ferramentas e técnicas para criação e manutenção de páginas na internet;
  2. Aplicação da estrutura de estilos em páginas Web;
  3. Estruturação de layouts de aplicações Web de acordo com os padrões W3C;
  4. Aplicação dos comportamentos em páginas Web utilizando JavaScript;
